# TaskFlow Pro 🚀
**MVP (Minimum Viable Product) - Ingeniería de Sistemas, 1er Semestre (UNIFRANZ)**

TaskFlow Pro es un gestor de tareas inteligente diseñado específicamente para ayudar a los estudiantes universitarios a organizar sus deberes por materias, realizar seguimiento de sus fechas de entrega y no perder de vista sus prioridades.

## 📌 Características de esta versión
Esta versión está restringida y optimizada para la carrera de **Ingeniería de Sistemas** en su **primer semestre**.

- **Carpetas por Materias:** El panel principal muestra automáticamente 7 carpetas exclusivas para el primer semestre (Programación I, Física I, Cálculo I, etc.).
- **Autocompletado Seguro:** Un menú estricto evita que el usuario cometa errores de tipeo al asignar tareas a una materia.
- **KPIs Visuales:** Seguimiento en tiempo real de tareas pendientes, completadas y totales.
- **Modo Claro/Oscuro:** Interfaz amigable para estudiar de día o de madrugada.

## 🛠️ Tecnologías Usadas
- **Backend:** Python (Flask, Werkzeug)
- **Frontend:** HTML5, CSS3, JavaScript (Feather Icons, Flatpickr)
- **Base de Datos:** Archivos JSON locales (Optimizados para prototipado rápido)

## 🚀 Despliegue en la Nube
El proyecto incluye todo lo necesario (`requirements.txt`, configuración de Flask) para ser desplegado instantáneamente en plataformas como **Render.com** o **PythonAnywhere** usando `gunicorn app:app`.
